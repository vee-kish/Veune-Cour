# Veune Studio · Portfolio by Vanessa Kinya

A modern, naturistic, and interactive single-page portfolio for **Veune Studio** by **Vanessa Kinya**, built with semantic HTML, modern CSS3, and vanilla JavaScript.

## Tech stack

- **HTML5**: Accessible, semantic sections for Home, Projects, About, and Contact.
- **CSS3**: Pastel palette, naturistic 3D animations (floating leaves, parallax hero, card tilt), responsive grid, micro-interactions, and smooth transitions.
- **Vanilla JavaScript**: Scroll-based navigation highlighting, parallax motion, interactive project cards, contact form validation, and reveal-on-scroll animations.

No frameworks or build tools are required.

## Getting started

1. Open `index.html` directly in your browser, or
2. Serve the folder with a simple static server (recommended for smooth scrolling and asset caching).

Example using Python:

```bash
cd "Doc 2"
python -m http.server 8000
```

Then visit `http://localhost:8000` in your browser.

## Customisation

- **Branding**: Update titles and text in `index.html` (hero, projects, about copy, footer story).
- **Colors & radius**: Tweak CSS custom properties at the top of `style.css`.
- **Animations & interactions**: Adjust keyframes and JavaScript behaviour in `style.css` and `script.js` (parallax depth, reveal thresholds, form messaging, etc.).

The codebase is intentionally small and modular to make edits easy.
